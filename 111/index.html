关于本项目

项目简介

本数据可视化平台基于Python Matplotlib的可视化案例开发，展示了图书采购数据和汇率走势的可视化分析。

数据来源

  • 图书采购数据：各地区对五类图书的采购情况
  • 汇率数据：2017年7月与2019年7月美元/人民币汇率走势

技术栈

HTML5 CSS3 JavaScript Chart.js

主要功能

  • 📊 交互式图表展示
  • 🔍 数据筛选与过滤
  • 📱 响应式设计
  • 📈 多种图表类型
  • 💾 数据导出功能
